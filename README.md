# 🏫 Website SMA Sains Miftahul Huda (Unofficial)

Repository ini berisi *source code* website statis untuk SMA Sains Miftahul Huda. Website ini pada awalnya dikembangkan untuk keperluan pameran (Expo) pendidikan, dan kemudian dilanjutkan pengembangannya sebagai pemenuhan tugas Ujian Akhir Semester (UAS) mata kuliah Cloud Computing.

## 📖 Latar Belakang & Alur Cerita Proyek

Proyek ini memiliki perjalanan yang cukup unik dan terbagi ke dalam dua fase pengembangan:

### Fase 1: Pameran Expo Sekolah (April 2026)
Awalnya, website ini dibuat secara khusus atas permintaan adik saya untuk ditampilkan pada acara Expo di SMA Sains Miftahul Huda pada bulan April. Target audiens dari Expo ini adalah anak-anak SMP (calon siswa baru) beserta orang tua mereka. 
*   **Optimalisasi Layar Lebar (TV Layar Sentuh):** Saat Expo berlangsung, webs ite ini ditampilkan sebagai etalase digital pada layar berukuran besar (TV bantuan pemerintah). Oleh karena itu, *User Interface* (UI) dari website ini—seperti ukuran *font*, ikon, dan elemen tata letak—sengaja didesain lebih besar (*oversized*) agar mudah dibaca dan menarik perhatian pengunjung dari jarak jauh.
*   **Status Website:** SMA Sains Miftahul Huda sejatinya sudah memiliki website resmi. Website ini dibuat murni sebagai bentuk kreativitas, pembanding desain, dan sarana pameran visual interaktif tanpa menggunakan sistem *backend* kompleks (100% statis).

### Fase 2: Tugas UAS Cloud Computing (Juli 2026)
Pada bulan Juli, terdapat penugasan UAS Cloud Computing yang menginstruksikan pembuatan website statis dengan tema portofolio, pariwisata, atau organisasi sekolah. Karena struktur dasar proyek Expo ini sangat relevan dengan instruksi tugas, pengembangannya pun dilanjutkan. Pembaruan utama difokuskan pada penambahan kelengkapan halaman dan perapian struktur agar sepenuhnya memenuhi kriteria akademik yang diminta.

## 🎯 Pemenuhan Kriteria Tugas UAS

Website ini telah dirancang dan diperbarui untuk memenuhi seluruh instruksi utama tugas UAS Cloud Computing:
- [x] **Beranda (`index.html`):** Menampilkan wajah depan sekolah, informasi eksklusif program percepatan, dan navigasi utama.
- [x] **Profil (`profil.html`):** Berisi sejarah, visi & misi, identitas lengkap sekolah (termasuk NPSN), dan pesan sambutan Kepala Sekolah.
- [x] **Informasi Utama (`program.html` & `fasilitas.html`):** Penjelasan mendalam terkait program unggulan (Percepatan 2 Tahun, Kajian Kitab Kuning) serta fasilitas *boarding school*.
- [x] **Kontak (`kontak.html`):** Informasi alamat lengkap, kontak WhatsApp/Email resmi, dan integrasi interaktif Google Maps.
- [x] **Deployment:** Website di-*hosting* secara publik menggunakan layanan gratis GitHub Pages tanpa memerlukan langganan *cloud* berbayar atau domain khusus.

## 📸 Catatan Dokumentasi & Aset
Sebagian besar aset foto dan dokumentasi kegiatan yang ada di dalam website ini adalah dokumentasi **asli** dari lingkungan nyata SMA Sains Miftahul Huda. Namun, untuk menjaga estetika desain visual dan menutupi beberapa kekurangan dokumentasi asli pada bagian *card* ilustrasi, terdapat beberapa gambar pendukung yang diambil dari penyedia gambar bebas hak cipta (Unsplash).
