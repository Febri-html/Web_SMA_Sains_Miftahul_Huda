# 🛠️ Petunjuk Penggunaan Website

Website SMA Sains Miftahul Huda ini dibangun sebagai website statis dan telah di-deploy menggunakan layanan GitHub Pages. Berikut adalah panduan langkah demi langkah untuk mengaksesnya.

### 🌐 Cara Akses Secara Online (Live)
1. Pastikan perangkat Anda terhubung dengan koneksi internet yang stabil.
2. Buka *browser* web pilihan Anda (Google Chrome, Firefox, Microsoft Edge, dll).
3. Klik atau salin URL publik berikut ke *address bar*: 
   🔗 **[https://febri-html.github.io/Web_SMA_Sains_Miftahul_Huda/profil.html]**
4. Website akan langsung dimuat. Anda dapat menjelajahi seluruh fitur melalui menu navigasi *fixed* di bagian atas layar (Beranda, Profil, Program, Fasilitas, dan Kontak).

### 💻 Cara Akses Secara Offline (Lokal)
Karena website ini murni dibangun menggunakan HTML dan CSS statis, Anda tidak memerlukan instalasi *local server* (seperti XAMPP atau Node.js) untuk menjalankannya.
1. Unduh *source code* dengan menekan tombol **Code** > **Download ZIP** pada *repository* ini.
2. Ekstrak file ZIP tersebut di perangkat Anda.
3. Buka folder hasil ekstraksi, lalu cari file utama bernama `index.html`.
4. Klik dua kali pada file `index.html` tersebut, dan website akan otomatis berjalan secara *offline* melalui *browser* bawaan Anda.
